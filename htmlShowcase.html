1. Semantics with New Structural Elements

HTML5 <header>, <nav>, <section>, <article>, <aside>, and < footer> to provide a more meaningful structure to web pages.

<video> and <audio>

HTML5 includes native support for embedding audio and video content using the <audio> and <video> elements. This eliminates the need for third-party plugins like Flash.

HTML5 introduces new input types such as <input type="date">, <input type="email">, and <input type="number">, providing better user experiences and enabling browsers to offer native date pickers, email validation, and number input controls.
